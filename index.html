About My Journey

Hi, I'm Juliah Denise P. Bantugon, a student and aspiring frontend developer from Batangas State University.

I started learning web development because I enjoy creating things that people can actually use. Unlike some advanced portfolios you might see, mine reflects where I am right now in my learning journey - comfortable with the basics and excited to learn more.

My current focus is mastering HTML, CSS, and JavaScript before moving to frameworks. I build simple projects to practice what I learn, and I'm not afraid to Google solutions when I get stuck (it's part of the process!).

My Approach:

  • • Start simple, then add complexity
  • • Code every day, even if it's just 30 minutes
  • • Build projects that solve real problems
  • • Learn from mistakes - they're the best teachers
